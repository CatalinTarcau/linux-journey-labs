# Reflection

## What did I learn?

During this lab, I learned how to confidently navigate the Linux filesystem and gained a better understanding of the purpose of its main directories. I started understanding the role each one plays in a Linux system and why this knowledge is essential for a future DevOps Engineer.

## What surprised me?

I was surprised by how well organized the Linux filesystem is. Every directory has a specific purpose, which improves maintainability, security and system organization. Understanding this structure made me realize that Linux is designed to separate responsibilities rather than storing everything in a single location.

## Where could this knowledge be useful in a real DevOps environment?

Understanding the Linux filesystem is fundamental when troubleshooting servers. Knowing where configuration files, binaries and log files are stored allows a DevOps Engineer to quickly investigate incidents, diagnose problems and perform system administration tasks more efficiently.
